--- layout: docs-item lang: en title: "sata-image" syntax: "sata-image($image, $display: false)" excerpt: "

Set background and width/height property of container.

" related: en: /en/docs/utils/sata-image.html ru: /ru/docs/utils/sata-image.html ---

{{page.syntax}}

{{page.excerpt}}

$image - url of image
$display - display css property (if it false, not set display property)

Examples

Show image 11932.jpg (282x158)

Source

{% highlight css %} #sata-image-example { @include sata-image("11932.jpg"); } {% endhighlight %}

Result

{% highlight css %} #sata-image-example { background: url('/assets/img/11932.jpg?1365412404'); width: 282px; height: 158px; } {% endhighlight %}
