Sata-scss

Библиотека mixin'ов для Sass(Compass)

Project maintained by satahippy Hosted on GitHub Pages — Theme by mattgraham

Введение

Библиотека mixin'ов для Sass(Compass). Модули:

  • all - включает все модули
  • utils - градиенты, изображения
  • lists - списки элементов (улучшеный Compass Lists)

Использование

Сначала надо установить Compass. Инструкция

Просто скопируйте файлы в директорию вашего проекта. Затем подключите те файлы (модули), которые вам нужны ("all" для подключения всех модулей). Вот так: {% highlight css %} @import "sata-scss/all"; {% endhighlight %}
