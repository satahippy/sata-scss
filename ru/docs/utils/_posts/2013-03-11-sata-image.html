--- layout: docs-item lang: ru title: "sata-image" syntax: "sata-image($image, $display: false)" excerpt: "

Устанавливает фон(изображение), высоту и ширину элемента.

" related: en: /en/docs/utils/sata-image.html ru: /ru/docs/utils/sata-image.html ---

{{page.syntax}}

{{page.excerpt}}

$image - url изображения
$display - css свойство display (если false, свойство не устанавливается)

Примеры

Показываем изображение 11932.jpg (282x158)

Код

{% highlight css %} #sata-image-example { @include sata-image("11932.jpg"); } {% endhighlight %}

Результат

{% highlight css %} #sata-image-example { background: url('/assets/img/11932.jpg?1365412404'); width: 282px; height: 158px; } {% endhighlight %}
